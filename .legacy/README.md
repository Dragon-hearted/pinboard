# Pinboard Legacy (web app)

This code ran the web-based Pinboard (React client + Hono server) until the
terminal rewrite. It is preserved here for rollback reference only and is not
wired into the current Pinboard runtime. Expect it to be deleted after one
release cycle of the terminal app, once TUI parity has been proven in daily
use (user approval required before removal).

See `specs/pinboard-terminal-rewrite.md` in the monorepo root for context on
the migration.
